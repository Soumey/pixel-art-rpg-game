# pixel-art-rpg-game
Pixel art rpg game with turn-based fight system <br />
What was used to create the game:<br />
HTML,CSS,JS<br />
Howler.js - sounds<br />
GSAP - animations<br />
Tiled - map and hitboxes<br />
Aseprite - assets<br />
